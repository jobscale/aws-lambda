# example

## versions

node v6.11.2
npm 3.10.10

### Getting source

```bash
$ git clone https://github.com/jobscale/xxx.git
$ cd xxx
```

### Setup

```bash
$ npm install
```

### Build module

```bash
$ npm run build
```

### Upload module

```bash
$ npm run zip
```

### Run module

```bash
$ npm run node
```
